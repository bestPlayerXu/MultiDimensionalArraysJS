# Multi Dimensional Arrays

```
const MultiDimensionalArrays = require('MultiDimensionalArrays');
var multiDArray = new MultiDimensionalArrays([2, 5, 4]);          //x length: 2; y length: 5; z length: 4
multiDArray.set('value', [1, 0, 2]);                              //set(value, point)
console.log(multiDArray.get([1, 0, 2]));                          //get(point)
```
